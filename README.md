## About the Project

This is a simple Python project that recommends items based on the user's interests. The user enters an interest such as movies, music, books, or coding, and the program displays a few recommendations using basic Python logic.

The main purpose of this project is to practice Python programming concepts like user input, conditional statements, lists, and loops.

---

## Features

* Takes input from the user
* Matches the user's interest
* Displays recommended items
* Easy to understand and beginner friendly
